Политика конфиденциальности

Дата вступления в силу: 1 марта 2026 г.

Capsule («мы», «наше», «приложение») — это приложение для отслеживания менструального цикла. Мы серьёзно относимся к защите ваших персональных данных.

1. Какие данные мы собираем

  • Данные аккаунта: email-адрес, никнейм (при регистрации)
  • Данные о здоровье: даты менструаций, симптомы, настроение, заметки
  • Данные медикаментов: названия и расписание приёма витаминов/лекарств
  • Технические данные: тип устройства, версия ОС, идентификатор устройства

2. Как мы используем данные

  • Для работы основных функций приложения (трекинг цикла, прогнозы, статистика)
  • Для генерации AI-советов на основе ваших симптомов и фазы цикла
  • Для синхронизации данных между устройствами (при наличии аккаунта)

3. AI-функции и Anthropic

Для генерации персонализированных советов приложение использует Claude AI от Anthropic. При использовании AI-функций:

  • Данные о симптомах и фазе цикла отправляются на серверы Anthropic
  • Anthropic не сохраняет персональные данные пользователей
  • AI-советы не являются медицинскими рекомендациями

4. Хранение данных

Основные данные хранятся локально на вашем устройстве. При регистрации аккаунта данные также хранятся на серверах Supabase (облачная платформа с шифрованием).

5. Передача данных третьим лицам

Мы не продаём и не передаём ваши персональные данные третьим лицам, за исключением:

  • Anthropic (Claude AI) — для генерации AI-советов
  • Supabase — для хранения и синхронизации данных аккаунта
  • RevenueCat — для управления подписками
  • Apple/Google — для обработки платежей

6. Безопасность

Мы используем шифрование при передаче данных (HTTPS/TLS). Приложение поддерживает биометрическую защиту (Face ID / Touch ID).

7. Ваши права

  • Экспорт всех данных (PDF или JSON-бэкап)
  • Полное удаление аккаунта и всех данных
  • Отзыв согласия на использование AI-функций

8. Дети

Приложение не предназначено для лиц младше 13 лет. Мы сознательно не собираем данные детей.

9. Изменения политики

Мы можем обновлять эту политику. О существенных изменениях вы будете уведомлены через приложение.
